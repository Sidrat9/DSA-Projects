# Snake Game

## Project Name: "Snake Game"
### Description:
A simple snake game using Python and Tkinter.

### DSA Concepts Used:
- Arrays
- Loops
- Event Handling

### How to Play:
- Use arrow keys (⬆️⬇️⬅️➡️) to control the snake.
- Eat food to grow and increase your score.
- Avoid colliding with walls and yourself.
